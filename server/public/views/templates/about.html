Welcome to Last Weather Yesterday!

This is the result of two weeks of solo work as part the Full Stack Software
Engineering Certification course from Prime Digital Academy.

Last Weather Yesterday is a weather history tracking tool. With an account,
you can choose to track weather data up to three zip codes. When you enter
a zip code, we save it in our database and then every hour on the hour, we
grab current weather data for that zip code from the Open Weather Map API
and store it in our database so that you can access an hour by hour weather
history for any location (in the US) of your choosing.

In addition to the weather data we save for you, you can leave yourself notes
about what you did on a given day or supplement the Open Weather Map
data with your own observations.
