Last Weather Yesterday

Last Weather Yesterday is a weather history tracker! It draws current weather data from
the Open Weather Map API for any zipcode stored in the database every hour on the hour and
then stores that data for you to reference whenever you'd like.
Create an account and track weather for up to three different zipcodes in the US.
